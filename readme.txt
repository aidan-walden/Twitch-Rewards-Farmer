INSTALLATION DIRECTIONS
1. Insatll Node.JS from https://nodejs.org/en/ (you want whichever version says "Recommended for most users")
2. Open command prompt and type "npm install puppeteer" (no quotes)

SETUP DIRECTIONS

0. Make sure your Blizzard and Twitch accounts are linked
1. Go to C:\Program Files (x86)\Google\Chrome
2. Copy Application folder and paste it in this directory
3. Create a txt file called streams (case sensitive)
4. Paste the stream schedule from the Blizzard blog post into streams.txt (see help.png) and save
5. Open run.bat
